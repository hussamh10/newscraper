Client quick start

On a different laptop:

git clone https://github.com/hussamh10/newscraper.git
cd newscraper
python3 -m venv .venv && source .venv/bin/activate
pip install -r client/requirements.txt
python client/run_client.py --server http://obelix.cs.uiowa.edu:8765 --loop --client-id laptop-1

Optional flags: --source tribune, --batch-size 50, --concurrency 20, --client-id your-name

Note: this endpoint is HTTP on port 8000 unless you put a TLS reverse proxy in front.
